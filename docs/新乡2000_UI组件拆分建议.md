# 新乡2000 UI 组件拆分建议

**版本**：v0.1  
**日期**：2026-05-22

## 三、Godot 导入注意事项

| 项目 | 设置 |
|------|------|
| 纹理过滤 | `Nearest`（像素风必须，全局设置） |
| 纹理重复 | `Disabled` |
| 拉伸模式 | `viewport`（Project Settings → Display → Window → Stretch） |
| 拉伸比例 | `keep` |
| 视口尺寸 | 根据最终确认：可选 320×180(×4) 或 1280×720(×1) |
| NinePatch 边距 | 按上表标注值设置 |
| 按钮三态 | 建议用 Godot StyleBoxTexture 分别设 normal/hover/pressed |

### 像素字体

- 中文像素字体推荐：Zpix（免费开源）
- 英文/数字：可用同一字体或更紧凑的等宽像素字体
- 字号建议：标题 16px，正文 12px，小标注 8px（按像素风设计基准）

### K线图表

- 不使用切图，由程序 `_draw()` 或 Line2D 实时绘制
- 蜡烛宽度建议：回合K 8px，分时K 3px
- 网格线：1px `#0f3460` 虚线
- 颜色：涨=`#00ff88`，跌=`#ff4444`
