## 一、游戏概述

### 1.1 游戏名称
**《不要怕，是技术性调整！》**

### 1.2 游戏类型
经营模拟 + 肉鸽元素的2D游戏

### 1.3 游戏简介
游戏提供了完整的交易界面，有商战、策略等多种要素，玩家需要跟随剧情逐步解锁商业投资的各个领域并完成自己的交易传奇故事。

### 1.4 设定背景
00年的新乡，正是金融泡沫爆发之后的萧条期，但是随着隔壁大陆市场的发展，飞港的经济有快速复苏的趋势。

玩家角色：一个普通的小吃摊主，经营着家里猪脚饭生意。在一个平静的下午，隔壁搬来一个老头，自称"宝叔"，来自国外。他说有发财的机会，可以指导你，但是需要压上你的全部积蓄。

嗅到金钱气息的玩家，想要抓住机会进入金融市场大赚一笔。

## 二、核心定位

这是一款围绕**股价操控与利润兑现**的策略卡牌游戏。

玩家不是单纯"买低卖高"，而是要通过卡牌和资源调度，主动制造股价波动。

**核心公式：**
> 制造价格 → 操控情绪 → 兑现利润

## 三、核心资源系统

玩家拥有四种核心资源：**金钱、股价、筹码、市场情绪**。

### 3.1 金钱

金钱既是玩家的**操作资源**，也是玩家的**生命值**（双重意义：金钱 = 操作资源 + 生存血）。

**金钱的用途：**
- 买入股票
- 打出部分技能牌
- 制造利好情绪
- 维持股价
- 触发事件或舆论操作

**失败判定：** 如果玩家本金损失过多，或者现金归零，就会失败。

### 3.2 股价

股价是游戏中的**核心波动对象**。

- **初始股价：100**
- 玩家通过卡牌效果、事件和情绪影响股价
- 股价越高，玩家手中的筹码市值越高
- 但如果无法成功卖出兑现，高股价只是账面财富

**股价受三类因素影响：**
1. 卡牌效果（具体是否影响股价写在每张卡牌上，买卖操作本身不一定直接影响股价）
2. 上涨情绪与下跌情绪
3. 事件牌和系统随机事件

### 3.3 筹码

筹码代表玩家持有的**股票数量**。

- 通过买入获得筹码，通过卖出将筹码换回金钱
- **持仓市值 = 筹码数量 × 当前股价**
- 筹码不仅是资产，也是一种操作资源

**筹码可以消耗来：**
- 砸盘
- 降低市场情绪
- 制造恐慌
- 触发某些技能牌
- 以换取更有利的买入机会

**关键点：** 筹码不是越多越好，关键在于什么时候买、什么时候卖、什么时候用筹码影响市场。

### 3.4 市场情绪

市场情绪由两个数值组成：**上涨情绪**和**下跌情绪**。

**核心规则：**
> 上涨情绪 + 下跌情绪 = 100

**初始值：** 上涨情绪 50，下跌情绪 50

**情绪档位与倍率（暂定，非线性）：**

| 上涨情绪区间 | 下跌情绪区间 | 买入上涨倍率 | 卖出下跌倍率 | 说明 |
|------------|------------|------------|------------|------|
| 0~30 | 70~100 | ×0.5 | ×2 | 极度恐慌，买入难拉，卖出暴跌 |
| 30~50 | 50~70 | ×0.8 | ×1.5 | 偏空，买入效果弱，卖出影响大 |
| 50~70 | 30~50 | ×1.5 | ×0.8 | 偏多，买入容易拉涨，卖出影响小 |
| 70~100 | 0~30 | ×2 | ×0.5 | 极度狂热，买入暴涨，卖出难跌 |

> 注：此为暂定四档方案，后续需给出完整数值表（见《后续所需》）

**情绪的特性：**
- 低位上涨敏感度高
- 高位下降敏感度高
- 市场情绪总是偏向中性

**情绪的间接效果：**
- 上涨情绪高 → 小额买入也能造成较大涨幅，卖出后价格不容易崩
- 下跌情绪高 → 大额买入也可能拉不动，少量卖出就可能引发大跌

## 四、核心价格逻辑

游戏的价格变化主要来自三部分：

> 卡牌效果造成的价格变化 + 市场情绪造成的自然变化 + 商战对手盘造成的变化（不一定有商战对手）

### 4.1 卡牌效果影响价格

**重要原则：买入/卖出操作本身不一定导致股价涨跌。** 是否影响股价、影响多少，由每张卡牌的效果文本决定。

- 例：初始买入牌——花费总资金的10%购买筹码，**不影响股价**
- 例：升级后的买入+牌——花费总资金的30%购买筹码，**同时影响股价+3%**
- 例：预测牌（内幕消息）——直接控制股价上涨3%

买卖操作与股价变化不强制挂钩，这是本游戏的核心机制设计。

### 4.2 市场情绪影响价格变化

情绪通过倍率放大或缩小卡牌对价格的影响效果（参见3.4情绪档位表）。

### 4.3 自然波动

股价在没有卡牌和事件影响时，每回合会产生一个自然波动值，从正态分布中抽取：

> 波动值 ~ Normal(μ, σ)

**均值 μ 由当前市场情绪决定：**

> μ = (上涨情绪 - 50) / 50 × 3%

- 上涨情绪 = 50 时，μ = 0%（中性，无方向偏移）
- 上涨情绪 = 70 时，μ = +1.2%（偏向上涨）
- 上涨情绪 = 30 时，μ = -1.2%（偏向下跌）
- 上涨情绪 = 100 时，μ = +3%（极度偏向上涨）
- 上涨情绪 = 0 时，μ = -3%（极度偏向下跌）

**标准差 σ：** 待后续数值调试确认（详见《后续所需》）

**最终波动值限制在 [-3%, +3%] 之间（clamp）。**

### 4.4 事件影响开盘价

事件会有一个**倍率属性**，影响第二天的开盘价：
- 例：事件倍率+3% → 第二天开盘价 = 前一天收盘价 × 103%
- 例：事件倍率-3% → 第二天开盘价 = 前一天收盘价 × 97%

> 具体事件设计后续给出（见《后续所需》）

## 五、核心玩法循环

局内的主要玩法循环总结为以下步骤：

### 第一步：建仓（金钱 → 筹码）
- 使用买入牌消耗金钱获得筹码
- 玩家需要控制买入节奏，避免一开始就把现金耗尽

### 第二步：制造价格
玩家通过卡牌效果制造股价上涨：
- 使用有价格影响效果的卡牌（如升级买入牌、预测牌）推动股价
- 配合技能牌或事件牌制造上涨趋势
- 市场变热 → 上涨情绪增加 → 后续价格操作更有效

### 第三步：操控情绪
玩家可以主动改变上涨情绪和下跌情绪：
- 花钱拉情绪 → 股价更容易上涨，更不容易下跌
- 花筹码砸情绪 → 股价更难上涨，更容易下跌

### 第四步：兑现利润
- 股价上涨后，玩家需要卖出筹码，将账面收益转化为金钱
- 玩家需要考虑：怎么维持价格？何时卖出最有利？

**核心总结：** 基本玩法循环中，玩家可以拿到三种资源：价格、资金和场内情绪，玩家需要做的是通过合适的技能与道具策略，把其他两种资源向资金转换。

## 六、游戏结构与回合系统

### 6.1 时间结构

游戏时间只有**三个层级**：

| 层级 | 说明 |
|------|------|
| **关卡** | 一局游戏可包含多个关卡（当前Demo只做一关） |
| **天** | 每关5个交易日（周一到周五） |
| **回合** | 每天10个回合 |

- 没有小时、分钟等概念
- 5天结束后结算金钱

### 6.2 K线显示机制

- 每天最多10根K线，一根K对应一个回合
- 加入**分时K**来展示每张牌对价格造成的影响
- 分时K的结果结算为一根**回合K**
- 回合K展示玩家每个回合对价格造成的影响
- 分时K展示即时反馈，显示每张牌玩家操作对价格造成的影响，每张牌刷新
- 商战对手也同样适用于此K线机制

### 6.3 每回合事件
- 每个回合都可能会有系统随机事件产生，影响股市情绪
- 此时会有一些弹幕显示，增加气氛
- 每回合结算的时候都有可能有弹幕
- 市场情绪会每回合自动的小幅影响股价

## 七、回合流程

### 7.1 单日流程阶段

1. **开盘阶段** — 开盘价确定
2. **抽牌阶段** — 玩家抽牌
3. **行动阶段** — 玩家打出卡牌（每回合3个行动点）
4. **价格结算阶段** — 根据卡牌效果和情绪计算价格变化
5. **情绪结算阶段** — 计算情绪变化
6. 重复步骤2~5共10个回合
7. **收盘结算阶段** — 结算当日利润
8. **盘后商店阶段** — 一天结束后进入商店，购买/升级/删除卡牌

### 7.2 基本规则

1. **玩家做多盈利**（玩家的主要盈利方向）
2. **初始股价：100；初始情绪：上涨50/下跌50**
3. **第一回合** 给玩家6张牌，每个回合发两张牌，手牌上限10张
4. **初始卡组（12张）：** 3张买入牌 + 3张卖出牌 + 3张预测牌（内幕消息） + 3张情绪牌（大V吹票）
5. **卡组循环机制：** 类似杀戮尖塔——玩家使用完卡牌后进入等待区；抽完卡组后等待区的卡牌依次进入卡组
6. **行动点系统：** 默认玩家行动点为3，行动点不可累计，使用某些技能卡可以恢复或者增大行动点上限
7. **最后一天平仓机制：** 最后一天（周五）结束后，未卖出的筹码强制以当前股价 × 50% 折算为金钱
8. **抽牌伪随机（第一回合）：** 第一回合保底保证有1张买入牌、1张卖出牌、1张技能牌，其他随机抽取

### 7.3 打出牌的规则
- 手牌上限为**10张**
- 是否影响股价由卡牌效果决定，买卖操作与股价变动不强制挂钩

### 7.4 游戏开始
- 会有第一个背景事件，给今天的股价走势定一个基调

## 八、商战系统

### 8.1 商战触发
- 有个对手突然跳出来，进入商战模式
- 商战对手一局最多出现一次

### 8.2 商战模式变化
- 对手也会出牌影响股价，会对玩家的持仓有很大影响
- 商战对手**每回合打出一张牌，效果是降低3%的股价**（暂定，详见《后续所需》）
- 要收割玩家资金

### 8.3 商战策略
- 玩家可以快攻来打败对手，也可以无视
- 建议玩家尽快解决对手，不然会给自己增加难度
- 打败对手可以停止其每回合的股价打压
- 玩家也可以利用商战对手达到降低股价吸筹的目的

### 8.4 商战流程（来自流程图）
玩家买入 → 低价买入品种 → 玩家配合消息拉升价格 → 使价格达到目标位置 → 玩家开始砸盘出货 → 散户高喊救命

对抗环节：
- 遭遇对手买入恶意抬价
- 发动负面消息、市场谣言等技能造成短时间价格暴跌，拉爆对手止损
- 遭遇对手恶意压价做空
- 增加资金，召唤NPC好友等技能大批买入，拉爆空头

## 九、胜利与失败条件

### 9.1 新手关（第一关）

- **初始金钱：** 100,000
- **胜利目标：** 5天结束后金钱达到 120,000 即为成功
- **时间限制：** 5个交易日（周一到周五）
- 最后一天结束后，未卖出的筹码强制以当前股价 × 50% 折算为金钱

另一种胜利方式：把对手资金消耗光为赢（打败对手后，操控股价会容易一些，然后获利结余）

### 9.2 失败条件
- 本金亏完（现金归零）
- 收益不足（5天结束后金钱未达到目标120,000）

## 十、卡牌系统

### 10.1 核心原则

**买卖操作与股价变动不强制挂钩。** 每张卡牌的效果文本明确写明是否影响股价、影响多少。

### 10.2 卡牌分类

卡牌分为**三大类**：基础牌、技能牌、事件牌

#### 基础牌（买牌和卖牌）

买入牌：消耗总资金的一定百分比购买筹码。初始买入牌不影响股价；升级后的买入牌可能附带股价影响效果。

- **买入（初始）：** 消耗总资金的10%购买筹码（按当前股价计算可买股数），不影响股价
- **买入+（升级版）：** 消耗总资金的30%购买筹码，同时影响股价+3%

卖出牌：卖出当前持仓的一定百分比换回金钱。初始卖出牌不影响股价；升级后的卖出牌可能附带股价影响效果。

- **卖出（初始）：** 卖出当前持仓筹码的10%（按当前股价换算获得金钱），不影响股价

#### 技能牌
技能牌用于改变局势，是游戏策略深度的主要来源。

按功能分类：
- **A. 控制上涨情绪的技能牌**
- **B. 控制下跌情绪的技能牌**
- **C. 维持价格的技能牌**
- **D. 触发buff的技能牌**

已确认效果的技能牌：

| 卡牌名称 | 类型 | 效果 |
|---------|------|------|
| 大V吹票 | 情绪 | -5%资金花费，提升场内情绪值+5 |
| 内幕消息 | 预测 | 直接控制股价上涨3% |
| 救市 | 情绪 | 市场情绪20以下使用，使其到达50 |

> 其他技能牌（恐惧、场外援助、顺势而为、小加一笔等）效果待后续设计（见《后续所需》）
> "过牌"技能牌：效果为额外抽牌

#### 事件牌
事件牌用于影响当日市场环境。大幅额外影响股价或者影响市场情绪。影响的是下一个回合的股价，但不消耗资金。

事件牌功能类型：
- 刷新当日事件
- 指定利好事件（业绩超预期，重大合作，政策扶持，公司回购，机构调研，行业爆发）
- 指定利空事件（业绩暴雷，监管调查，大股东减持，行业利空，舆论危机，资金出逃）
- 改变事件强度
- 延长事件持续时间
- 取消负面事件

> 具体事件牌效果待后续设计（见《后续所需》）

### 10.3 初始卡组构成（Demo第一关）

| 卡牌 | 数量 | 效果 |
|------|------|------|
| 买入 | 3张 | 消耗总资金10%购买筹码，不影响股价 |
| 卖出 | 3张 | 卖出当前持仓筹码的10%换回金钱（按当前股价换算），不影响股价 |
| 内幕消息 | 3张 | 预测类，直接控制股价上涨3% |
| 大V吹票 | 3张 | 情绪类，-5%资金花费，提升场内情绪值+5 |
| **合计** | **12张** | |

### 10.4 卡牌循环机制
- 玩家使用完卡牌之后，对应卡牌进入**等待区**
- 玩家抽完卡组之后，等待区的卡牌依次进入卡组（类似杀戮尖塔的弃牌堆洗回抽牌堆机制）

## 十一、玩法策略示例

### 策略一：拉升兑现流
先买 → 再制造利好事件 → 股价上涨后逐步卖出
> 买入 → 财经头条 → 利好事件 → 股价上涨 → 护盘 → 分批卖出

### 策略二：情绪泡沫流
用技能牌制造市场狂热，让股价自己涨
> 买入 → 大V喊单 → 社交平台刷屏 → 赚钱效应 → 散户追涨 → 高位卖

### 策略三：保险流派
> 买足保险 → 使用收益放大卡牌 → 股价降低无限回血

### 策略四：信用流派（灵感来源蚂蚁金融）
> 使用道具卡牌触发无限质押 → 把一部分筹码质押n次 → 一回合借到足够的钱 → 大单砸盘获取超额收益

## 十二、玩家每日策略选择（来自流程图）

玩家每天可以选择当日策略方向：

### 做多路线
打压价格 → 用有限资金购买筹码 → 拉升价格 → 维持高价平仓

### 做空路线
哄抬价格 → 做空筹码 → 打压价格 → 维持低价平仓

### 商战Boss路线
突遇商战对手(Boss) → 对手和你做相反方向 → 需要把对手资金耗完 → 平仓大赚一笔

> 注：所有操作通过卡牌完成，没有单独的"做多/做空"按钮

## 十三、关卡结构与资源流动

### 13.1 关卡类型

游戏包含三种关卡类型：**新手关卡、普通关卡、挑战关卡**
（当前Demo只做新手关卡）

### 13.2 新手关卡（详细流程）
