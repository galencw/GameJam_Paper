# 新乡2000 UI 界面设计方案

**版本**：v0.1  
**日期**：2026-05-22  
**目标引擎**：Godot 4.x  
**基准分辨率**：1280×720 (16:9)  

> 本文档基于《传奇交易员--新乡2000_完整策划文档》及 image4/image7 线框图设计，不沿用旧版飞港2000界面。

## 阶段2：布局方案

### 一、交易界面布局

#### 1.1 整体分区

基于 image4 下方线框图和 image7 详细标注，移除道具栏和保险后重新分配右侧空间：

```
┌─────────────────────────────────────────────────────────────────┐
│ [A] 顶部信息栏                                                    │
```

#### 1.2 各区域详细说明

**[A] 顶部信息栏** — 全宽横条，高约 40px

从左到右排列：
1. **天数**：`周三 第3/5天` — 白色文字
2. **回合数**：`第 7/10 回合` — 白色文字
3. **弹性空白**
4. **市场情绪按钮**：金色底，点击可展开情绪详情
5. **突发事件按钮**：金色底，有事件时高亮闪烁
6. **速度控制**：暂停/播放/快进 三个圆形按钮

> 行动力已移至底部操作区[F]手牌区上方，便于玩家出牌时查看。

**[B] 四色金钱柱状条** — 左侧竖条，宽约 40px

基于 image4/image10 线框，从上到下四色分段：
- 蓝色段：安全资金区（资金充裕）
- 绿色段：正常区
- 黄色段：警戒区
- 红色段：危险区（接近归零）

当前现金在柱状条上用白色标记线标注位置。柱状条是视觉化的资金健康度指示器，精确数字在右侧面板显示。

**[C] K线图表区** — 中央主区域

分为上下两个图表，各占约一半高度：

- **回合K（上）**：黑底 + 网格，最多10根K线（一天10回合），红涨绿跌
  - 标题："回合K"
  - 弹幕层叠加在此区域上方飘过
  
- **分时K（下）**：黑底 + 网格，展示当前回合内每张牌的即时价格影响
  - 标题："分时K"
  - 每打出一张卡牌时实时刷新

**[D] 右侧数据面板** — 右侧竖向面板，宽约 200px

道具栏和保险已移除，此面板专注于核心交易数据，从上到下排列四个信息块：

1. **股价块**：股票名 + 当前价格 + 涨跌额 + 涨跌幅(%) + 涨跌颜色
2. **现金块**：当前现金数字（金色大字）
3. **持仓块**：持仓股数 + 当前市值
4. **盈亏块**：总盈亏金额 + 盈亏百分比 + 涨跌颜色

**[E] 情绪条** — K线区与手牌区之间的横向分隔带

基于 image6 参考设计：
- 左绿右红的双向水平条
- 左侧显示上涨情绪数值（绿色），右侧显示下跌情绪数值（红色）
- 分界线位置随情绪比例移动
- 上涨情绪+下跌情绪=100
- 点击"市场情绪"按钮可弹出详细倍率信息（如 image6 中的气泡提示）

**[F] 底部操作区** — 底部区域，高约 180px

三列布局：

- **左列 — 商战对手区**：
  - 对手头像（未出现时灰色/隐藏）
  - 对手金钱条（竖向小条）
  - "卡组"按钮（查看对手牌组，可选）

- **中列 — 手牌区**：
  - **行动力显示**（手牌区正上方居中）：3个图标（●=可用，○=已消耗），如 `●●○`，出牌时实时减少
  - 横排卡牌，默认显示5张
  - 手牌上限10张：超过5张时底部出现翻页箭头或缩小间距
  - 卡牌可点击选中 → 弹出详情 → 确认打出
  - 打出卡牌消耗行动力（行动力图标实时更新）

- **右列 — 玩家区**：
  - 玩家头像
  - 玩家金钱条（竖向小条，与左侧四色柱联动）
  - "卡组"按钮（查看自己完整牌组）

#### 1.3 交互流程在界面上的体现

```
回合开始 → 顶部"回合数"更新 → 抽牌动画（牌从卡组飞入手牌区）
→ 玩家从手牌区选牌打出 → 分时K实时刷新 → 行动力减少
→ 行动力用完或玩家选择结束 → 价格结算（回合K新增一根）
→ 情绪结算（情绪条动画变化）→ 下一回合
→ 第10回合结束 → 收盘结算 → 跳转商店界面
```

#### 2.2 各区域详细说明

**[G] 顶部信息栏** — 全宽横条

- 标题："盘后商店"
- 天数进度：`第3/5天结束`
- 当前现金：`¥85,200`（金色大字，实时更新，购买后立即扣减）

**[H] 当日结算摘要** — 左上区域

展示今日交易成果：
- 今日开盘价 / 收盘价 / 涨跌幅
- 持仓变化（是否有强制平仓）
- 今日盈亏金额
- 累计总资金
- 胜利目标提示：`目标 ¥120,000 | 当前 ¥135,120 ✓` 或 `目标 ¥120,000 | 还差 ¥15,000`

**[I] 当前牌组预览** — 右上区域

- 以缩略图形式展示玩家完整牌组
- 按类型分组：基础牌 / 技能牌 / 事件牌
- 每张卡牌显示名称和数量
- 牌组总数
- 点击可查看卡牌详情

**[J] 商店操作区** — 下半部分，三个标签页

通过顶部标签切换：

**买卡标签页**：
- 横排展示可购买的新卡牌（3-5张商店刷新的卡牌）
- 每张卡显示：卡牌名 + 缩略图 + 价格 + [购买]按钮
- 选中卡牌时弹出详细效果预览
- 买不起的卡牌灰色显示

**升级标签页**：
- 展示玩家牌组中可升级的卡牌
- 每张显示：原版卡牌 → 升级后卡牌 + 效果对比 + 价格 + [升级]按钮
- 例：`买入 → 买入+`（10%→30%，新增股价+3%）

**删卡标签页**：
- 展示玩家牌组中所有卡牌
- 每张可点击选中 → 显示删除价格（¥1,000起递增）→ [删除]按钮
- 删除确认弹窗："确定删除[买入]牌？花费¥1,000"

**[离开按钮]** — 底部居中

- "离开商店，进入下一天 →"
- 最后一天(周五)改为："结束本周，进入结算 →"

#### 2.3 商店交互流程

```
收盘结算画面（自动展示） → 进入商店界面
→ 玩家浏览买卡/升级/删卡（可自由切换标签，操作不限次数，只要有钱）
→ 点击"离开商店" → 确认弹窗"确定离开？"
→ 进入下一天开盘 或 周五结束进入最终结算
```
